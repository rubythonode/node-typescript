Global packages
=============
```
Install NodeJS v8.9.0 (LTS - Recommended For Most Users): https://nodejs.org/en/download/
$ sudo npm install -g typescript nodemon mocha nyc codecov
```

Install dependencies
=============
```
$ npm install
```

Build code
=============
```
$ npm start
```

Testing
=============
```
npm test
```
